**The Minimum Wage Legislation on Gig Workers**

**Author:** Erya Ouyang

**Status:** Data Collection

**Abstract:**

This project examines how changes in minimum wage legislation affect labor supply, earnings, and platform participation among gig economy workers. Using causal inference methods and large-scale platform data, we study how wage floor policies reshape the gig economy landscape and worker welfare.

**Keywords:** Gig Economy, Minimum Wage, Labor Economics, Platform Markets, Causal Inference
